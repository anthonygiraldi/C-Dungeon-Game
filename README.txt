------------------------
HOW TO COMPILE AND RUN
------------------------
Compile: g++ -std=c++17 gameDriver.cpp map.cpp inventory.cpp monster.cpp party.cpp status.cpp
Run: ./a.out

------------------------
DEPENDENCIES
------------------------
inventory.h, map.h, monster.h, party.h, and status.h must be in the same directory as the cpp
files in order to compile.

------------------------
ABOUT THIS PROJECT
------------------------
This project is a text based dungeon RPG where the player must clear five different rooms while keeping their
party alive using food. There are NPC's around the randomly generated map where you can buy and sell items. There
are mosnters to defeat and riddles to solve.
